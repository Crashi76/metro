Setup Info:
===========
This version of the welcome app is only compatible with upcoming ownCloud Version 8.1 or later!

Installation:
=============
Download the zip file and rename folder from welcome-master to welcome! Upload the app to your apps directory and activate it on the apps settings page!
Go to the adminpanel click on the leftside bar on "Themes Manager Switcher" choose on "Default App on Start" the welcome app as active!

Additional Feature:
===================
On the admin panel is now under "Themes Manager Switcher" a selectbox available where the admin can activate a theme!
